;
; Logoi is a minimalist "superdialect" of Prolog & Lisp designed for rapid natural language programming.
;
; Under the hood, Logoi is rewritten to LIPS Scheme [JavaScript] and SWI-Prolog [WASM].
;
; Why? Because the only thing better than Prolog or Lisp is Prolog and Lisp.
;
; Logoi does not desire to achieve an "additive glueing" of two halves into a haphazard whole.
;
; Logoi desires instead to achieve a "genetic grafting" of two paradigms (logical/functional) into a hybrid third.
;
; In short: "Lisp sans parens, et Prolog sans Prolog!"
;
; Logoi lives in pursuit of that sweet state of flow which follows from an unburdened creativity.

;
; Part II: Syntax Highlighting
;
; Logoi has three basic building blocks for the entire language:
;
; 1. Constants are things that cannot change over time [123, "Antarctica", true, etc.]
; 2. Variables are things that can change over time [current-score, X, is-it-midnight?, etc.]
; 2. Functions are things that bundle constants and variables together into abstract processes.
;
; On this page, these three building blocks are colored in this way:
;

<constants are sky-blue>

<variables are white>

<functions are lavender>

;
; Part VI: Logic Programming
;
; Prolog is a powerful logic programming language with a steep learning curve.
;
; Logoi translates your logic for you so you can use Prolog without learning Prolog.
;

english
`Socrates is a man.`
`The plural of man is men.`
`All men are mortal.`

`Is Socrates mortal?`

; That's not Prolog, that's English!

prolog
`'{1} is a {2}.'('Socrates', 'man').`
`'The {1} of {2} is {3}.'('plural', 'man', 'men').`
`'All {1} are {2}.'('men', 'mortal').`

`'All {1} are {2}.'(X, 'mortal'),
'The {1} of {2} is {3}.'('plural, Y, X),
'{1} is a {2}.'('Socrates', Y).`

; Ahh, there's the Prolog! I'll admit the English version is easier to read.

english
`Cain is a man.`
`Abel is a man.`

`Eve is a parent of Cain and Abel.`

`{1} is a brother of {2} if
{1} is not {2},
{1} is a man,
{2} is a man,
{3} is a parent of {1}, and
{3} is a parent of {2}.`

`Is Cain a brother of Abel?`

; Can I see the Prolog version of this English logic?

prolog
`'{1} is a {2}.'('Cain', 'man').`
`'{1} is a {2}.'('Abel', 'man').`

`'{1} is a {2} of {3}.'('Eve', 'parent', 'Cain').`
`'{1} is a {2} of {3}.'('Eve', 'parent', 'Abel').`

`'{1} is a brother of {2}'(A, B) :-
A \= B,
'{1} is a {2}.'(A, 'man'),
'{1} is a {2}.'(B, 'man'),
'{1} is a {2} of {3}.'(C, 'parent', A),
'{1} is a {2} of {3}.'(C, 'parent', B).`

`'{1} is a brother of {2}.'('Cain', 'Abel').`

; It's nice to know I have both options. Is English the only human language Logoi knows?
