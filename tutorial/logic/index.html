; "An Informal Review of Logic Programming"
;
; Imagine a recipe for a delicious meal: Ingredients + Instructions = Result
;
; Traditional programming is like cooking by reading only the Instructions. ;
; Logic programming is like cooking by:
; 1. Looking at the Ingredients, then
; 2. Looking at the Result, then
; 3. Logically deducing how to turn the Ingredients into the Result without looking at the Steps.
;
; Fortunately, for the average ape, the logical deduction process above is automated by Prolog.
