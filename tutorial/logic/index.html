; An Informal Review of Logic Programming
;
; Imagine a recipe for a delicious meal: Ingredients + Steps + Result::Photo
;
; Traditional programming is like cooking by only reading the Steps of the recipe, without looking at either the Ingredients or the Result.
;
; Logic programming is like looking at the Ingredients, then looking at the Result, then logically deducing how to turn the Ingredients into the Result without reading the Steps.
;
; Fortunately for the average ape, the above process of logical deduction is automated by Prolog.
