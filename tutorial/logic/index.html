; An Informal Review of Logic Programming
;
; Imagine a recipe for a delicious meal: Ingredients + Steps + Result::Photo
;
; Traditional programming languages are like cooking by only reading the Steps of the recipe, without reading the Ingredients or the Result.
;
; Logic programming languages are like looking at the Ingredients, looking at the Result, then logically deducing how to turn the Ingredients into the Result without reading the Steps.
;
; Fortunately for the average ape, the process of logical deduction is automated.
