; Logoi is a minimalist "superdialect" of Prolog & Lisp designed for rapid natural language programming.
;
; Under the hood, Logoi is rewritten to LIPS Scheme [JavaScript] and SWI-Prolog [WASM].
;
; Why? Because the only thing better than Prolog or Lisp is Prolog and Lisp.
;
; Which led to the design motto: "Lisp sans parens, et Prolog sans Prolog."
;
; Because we already think in natural languages, so we should be coding in those natural languages.
