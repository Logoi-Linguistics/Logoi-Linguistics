; Logoi Tutorial, Part I: Human Commentary
;

;
; There are two kinds of human commentary in Logoi:
;
; 1. Editorial (;)
; 2. Logical (%)
;
; Everything to the right of a semicolon (;) is considered editorial commentary, which Logoi ignores.
;
; Everything to the right of a percentile (%) is considered logical commentary, which Logoi interprets.
;

% Socrates is a man.
% All men are mortal.
% The plural of man is men.

% Is Sorates mortal?
